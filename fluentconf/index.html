Your Script Just Killed My Site
High Performance Snippets

Steve Souders, Google

Your script just killed my site

  • Identify the SPOFs on your site: ads, widgets, etc
  • If you're building widgets that are going to be used on 3rd party websites, don't make them SPOF
  • <script src='script.js'> = SPOF
    Entire content blocked until they finish loading
  • Test your site and identify the weakest link
    Webpagetest.com to do so. Think about scripts blocked by firewalls (ex, China)

Your script just killed my site (cont.)

Mobile Device APIs for Fun and Profit

  • Mobile web is NOT only media queries. That's just a way of detecting screensize.
  • Mobile web != Responsive Web Design
  • Most of Facebook's mobile traffic comes from web.
  • So... let's take advantage of the Device APIs.

Device APIs (cont.)

  • Client-side APIs that let's us interact with the device hardware.
  • We can make applications that can jump outside the browser and enhance the user experience.

Device APIs (cont.)
